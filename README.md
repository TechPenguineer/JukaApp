# Juka App

## Run Juka Programming Language in a GUI.

![jukaApp](https://user-images.githubusercontent.com/11934545/179396687-e7a7008e-0816-46f5-8648-bce4e3217ace.png)

Platforms Supported:

- Android and AndroidTV
- iOS (iPhone and iPad)
- Macintosh (MacCatalyst)
- Tizen (Samsung)
- Windows App
